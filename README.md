# Bowling-Game

This is the Bowling-game-kata kata in TypeScript.
- https://kata-log.rocks/bowling-game-kata

## Getting started

Install dependencies

```sh
npm install
```

## Running app

_You may need to install `ts-node`_

## Running tests

To run all tests

### Jest way

```sh
npm run test:jest
```

To run all tests in watch mode

```sh
npm run test:jest:watch
```
